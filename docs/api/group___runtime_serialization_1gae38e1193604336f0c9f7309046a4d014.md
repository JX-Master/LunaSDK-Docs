# Luna::deserialize

```c++
RV deserialize(typeinfo_t type, void *inst, const Variant &data)
```

Deserializes one value. 



## Parameters
* *in* **type**

    The type of the instance. 

* *in* **inst**

    The instance data. 

* *in* **data**

    The serialized data used for deserialization.
