# Luna::logv

```c++
void logv(LogVerbosity verbosity, const c8 *tag, const c8 *format, VarList args)
```

Logs one message. 



## Parameters
### verbosity
The log verbosity. 

### tag
The log tag. Used by the implementation to filter logs. 

### format
The log message format. 

### args
Arguments used to format the log message.
