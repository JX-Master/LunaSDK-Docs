# Luna::operator/

```c++
Float3x3 operator/(const Float3x3 &m1, f32 s)
```

Divides one scalar with one matrix (m1 / s). The scalar will be divided from every component of the matrix. 



## Parameters
* *in* **m1**

    The matrix. 

* *in* **s**

    The scalar. 

## Return value
Returns the result matrix.
