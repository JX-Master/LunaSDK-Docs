# Luna::operator*

```c++
Float3x3 operator*(f32 s, const Float3x3 &m1)
```

Multiplies one matrix with one scalar. The scalar will be multiplied to every component of the matrix. 



## Parameters
* *in* **s**

    The scalar. 

* *in* **m1**

    The matrix. 

## Return value
Reutrns the result matrix.
