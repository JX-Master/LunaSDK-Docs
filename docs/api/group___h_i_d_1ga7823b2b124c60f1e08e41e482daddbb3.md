# Luna::HID::MouseButton

```c++
enum MouseButton : u8
{
    left= 0x01
    right= 0x02
    middle= 0x04
    function1= 0x08
    function2= 0x10
}
```

Specifies mouse button. These values can be bitwise-OR combined to select multiple mouse buttons if needed.
