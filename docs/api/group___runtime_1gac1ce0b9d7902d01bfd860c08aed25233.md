# Luna::VariantType

```c++
enum VariantType : u8
{
    null= 0
    object= 1
    array= 2
    number= 3
    string= 4
    boolean= 5
    blob= 6
}
```

Defines all possible types of one [Variant](class_luna_1_1_variant.md) object. 



## Remark
To fetch the type of one variant, call Variant::type
