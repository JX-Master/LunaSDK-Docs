# Luna::operator/

```c++
Float4x4 operator/(const Float4x4 &m1, const Float4x4 &m2)
```

Divides one matrix with another matrix (m1 / m2). The division is performed on every component of two matrices. 



## Parameters
* *in* **m1**

    The first matrix. 

* *in* **m2**

    The second matrix. 

## Return value
Returns the result matrix.
