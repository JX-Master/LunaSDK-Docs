# Luna::operator-

```c++
Float2 operator-(const Float2 &v1, const Float2 &v2)
```

Subtracts two vectors. 

This function performs the following operations: 
```
[Float2](struct_luna_1_1_float2.md) r;
r.x = v1.x - v2.x;
r.y = v1.y - v2.y;
return r;
```


## Parameters
* *in* **v1**

    The first vector. 

* *in* **v2**

    The second vector. 

## Return value
Returns the result vector.
