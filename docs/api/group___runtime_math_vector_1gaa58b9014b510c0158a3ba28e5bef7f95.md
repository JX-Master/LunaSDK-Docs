# Luna::length_squared

```c++
f32 length_squared(const Float2 &vec)
```

Computes the squared length of the vector. 

This function performs the following operations: 
```
return vec.x * vec.x + vec.y * vec.y;
```


## Parameters
* *in* **vec**

    The vector. 

## Return value
Returns the squared length of the vector.
