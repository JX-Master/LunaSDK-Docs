# Luna::Rect::Rect

```c++
Rect(_Ty offset_x, _Ty offset_y, _Ty width, _Ty height)
```

Constructs one rectangle. 



## Parameters
* *in* **offset_x**

    The X position of the rectangle. 

* *in* **offset_y**

    The Y position of the rectangle. 

* *in* **width**

    The width of the rectangle. 

* *in* **height**

    The height of the rectangle.
