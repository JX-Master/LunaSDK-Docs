# Luna::operator-

```c++
Float4x4 operator-(const Float4x4 &m1, f32 s)
```

Subtracts one scalar from one matrix (m1 - s). The scalar will be subtracted from every component of the matrix. 



## Parameters
* *in* **m1**

    The matrix. 

* *in* **s**

    The scalar. 

## Return value
Returns the result matrix.
