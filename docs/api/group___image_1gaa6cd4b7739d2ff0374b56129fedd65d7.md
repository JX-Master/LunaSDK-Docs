# Luna::Image::read_dds_image

```c++
R< DDSImage > read_dds_image(const void *data, usize data_size)
```

Reads DDS image data from DDS image file data. 



## Parameters
* *in* **data**

    The image file data. 

* *in* **data_size**

    The size of the image file data in bytes. 

## Return value
Returns the read DDS image.
