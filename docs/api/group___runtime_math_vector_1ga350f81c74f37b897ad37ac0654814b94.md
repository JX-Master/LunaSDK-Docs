# Luna::length

```c++
f32 length(const Float3 &vec)
```

Computes the length of the vector. 

This function performs the following operations: 
```
return sqrt(vec.x * vec.x + vec.y * vec.y + vec.z * vec.z);
```


## Parameters
* *in* **vec**

    The vector. 

## Return value
Returns the length of the vector.
