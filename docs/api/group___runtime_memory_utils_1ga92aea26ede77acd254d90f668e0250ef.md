# Luna::bit_set

```c++
void bit_set(void *addr, usize bit_offset)
```

Sets the specified bit to 1. 



## Parameters
* *in* **base_addr**

    The address of the bit to offset from. 

* *in* **bit_offset**

    The number of bits shifted from the `base_addr`.
