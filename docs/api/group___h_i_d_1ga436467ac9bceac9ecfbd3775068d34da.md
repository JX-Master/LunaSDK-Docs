# Luna::HID::ControllerButton

```c++
enum ControllerButton : u32
{
    lthumb= 0x0001
    rthumb= 0x0002
    up= 0x0004
    down= 0x0008
    left= 0x0010
    right= 0x0020
    a= 0x0040
    b= 0x0080
    x= 0x0100
    y= 0x0200
    lb= 0x0400
    rb= 0x0800
    lt= 0x1000
    rt= 0x2000
    lspecial= 0x4000
    rspecial= 0x8000
}
```

Specifies controller buttons. These values can be bitwise-OR combined to select multiple mouse buttons if needed.
