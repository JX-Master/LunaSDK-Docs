# Luna::log

```c++
void log(LogVerbosity verbosity, const c8 *tag, const c8 *format,...)
```

Logs one message. 



## Parameters
* *in* **verbosity**

    The log verbosity. 

* *in* **tag**

    The log tag. Used by the implementation to filter logs. 

* *in* **format**

    The log message format.
