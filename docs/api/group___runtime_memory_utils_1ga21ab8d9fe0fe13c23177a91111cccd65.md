# Luna::memzero

```c++
void * memzero(void *dst, usize byte_count)
```

Clears the specified memory region to 0. 



## Parameters
* *in* **dst**

    The start address of memory region to clear. 

* *in* **byte_count**

    The size, in bytes, of the memory region to clear. 

## Return value
Returns the `dst` pointer.
