# Luna::Name::Name

```c++
Name(const String &str, usize pos, usize count)
```

Constructs one name from one substring of the provided string. 



## Parameters
* *in* **str**

    The name string. 

* *in* **pos**

    The first character used for the name. 

* *in* **count**

    The number of characters used for the name.
