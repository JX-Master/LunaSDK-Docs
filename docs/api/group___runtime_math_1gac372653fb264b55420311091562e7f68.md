# Luna::operator+

```c++
Float4x4 operator+(const Float4x4 &m1, const Float4x4 &m2)
```

Adds two matrices. The addition is performed on every component of two matrices. 



## Parameters
* *in* **m1**

    The first matrix. 

* *in* **m2**

    The second matrix. 

## Return value
Returns the result matrix.
