public LUNA_RUNTIME_API voiddebug_printf(const c8 * fmt,...)

Prints one debuf string to the host's debug console.

Parameters

  • fmt The formatting syntax of the output string.

  • args Arguments used to format the output string.

This function does not necessary output the debug string to stdout or stderr. To output string to stdout, use get_std_io_stream instead.
